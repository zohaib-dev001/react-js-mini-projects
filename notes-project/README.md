# Notes App (React JS)

A simple Notes App built with React JS that allows users to add and delete notes.  

##  Features

- Add new notes 
- Delete notes
- Dynamic rendering using `map()`
- React Hooks (`useState`)
- Component-based architecture

##  Concepts Used

- React Functional Components
- useState Hook
- Props (Parent to Child data flow)
- Event Handling
- List Rendering
- Component Reusability

##  Installation & Run
```bash
npm install
npm run dev
