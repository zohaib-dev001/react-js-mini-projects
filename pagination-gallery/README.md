# React Pagination Gallery

A fully responsive image gallery application built with React JS.  
This project fetches images from an external API and displays them with pagination support.

## Features

- Responsive Design
- API Data Fetching
- Pagination Functionality
- Loading State
- React Functional Components
- React Hooks
- Material UI Pagination
- Tailwind CSS Styling

## Technologies Used

- React JS
- Tailwind CSS
- Axios
- Material UI
- Picsum Photos API

##  Installation & Run

```bash
npm install
npm run dev
