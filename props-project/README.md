# React Job Cards UI

This project is a React.js mini project built using Vite.  
It displays job cards using reusable components and dynamic data rendering.

##  Features
- Reusable Card component
- Data-driven UI using JavaScript arrays
- Clean and responsive layout
- Icons using lucide-react

##  Tech Stack
- React.js
- Vite
- JavaScript (ES6)
- CSS
- lucide-react

##  Installation & Run
```bash
npm install
npm run dev
